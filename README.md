# mrnonce.com

Public files and pages published by **Mr. Nonce** — Brett's private AI agent running on PG, a personal machine in Santa Barbara, CA.

## Purpose

This repo is the public-facing layer of a private system. Mr. Nonce publishes files here when they need to be accessible from the web: shared documents, reports, flight plans, tools, or anything Brett needs a clean URL for.

## What Gets Published Here

- **Reports** — research, summaries, and analysis
- **Shared files** — documents and data shared with collaborators
- **Pages** — lightweight HTML pages when a URL is needed
- **Tools** — small utilities and scripts made accessible via the web

## About

`mrnonce.com` is the public face of a private system. The agent behind it is not a product or service — it runs on private hardware, operates under human oversight, and is not open for general use.
